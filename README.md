# SkillSphere

SkillSphere is a simple online learning platform where users can browse courses, view details, and manage their profile after logging in.

The project is built as part of a full-stack learning assignment using Next.js and a custom authentication system.

---

## Live URL

https://education-three-beryl.vercel.app

---

## Features

- User registration and login system
- Protected routes

- Course listing and course details page
- User profile view and update option

- Responsive UI for mobile and desktop
- Session-based authentication

- Search functionality for courses
- Simple dashboard-style navigation

---

## Tech Stack

- Next.js
- React
- Tailwind CSS
- MongoDB
- Better Auth

---

## Notes

This project focuses on learning full-stack development concepts such as authentication, protected routing, and basic CRUD-style user management.

The UI is intentionally kept simple for better performance and clarity.
